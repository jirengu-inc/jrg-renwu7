# 使用步骤
1. 获取本 repo 读写权限
2. `git clone git@github.com:jirengu-inc/jrg-renwu7.git`
3. `cd jrg-renwu7`
4. `git config --global user.name 你的名字`
5. `git config --global user.email 你的邮箱`
6. 开始写代码！
